## Запуск на linux
1. Сделать bash скрипт исполняемым файлом `$chmod +x ./venv_script`
2. Запустить скрипт `$./venv_script`
3. Активировать виртуальное окружение `$source venv/bin/activate`
4. Запустить парсер `$python3 main.py`
